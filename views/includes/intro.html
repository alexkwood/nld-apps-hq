These apps were built by web developer Ben Buckman of New Leaf Digital,
using node.js, express.js, and mongoDB.
All the code for these apps is on GitHub.
The apps were created for personal use, then this suite was built to master the node/express/mongo stack and as a showcase of production code.
Feel free to use them, but note that the code may be buggy, the apps might crash at any time, and your data may be lost. Use these apps entirely at your own risk.
You may login to the apps using your Facebook ID.
Enjoy!
